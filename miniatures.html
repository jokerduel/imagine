Miniatures


Details

These images were generated using the default FLUX.1 Kontext [dev] workflow (fp16) with the following prompt: "Transform the main character into a hand-painted 3D model using the same style and colors. Ignore all background elements. Place on a black stand, photorealistic wood table"

The source images were the restored creature and Magi cards available on this site. Each image was only generated once, and if it had major defects it was removed.
